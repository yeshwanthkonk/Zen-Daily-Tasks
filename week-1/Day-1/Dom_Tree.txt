1. When a web page is loaded, the browser creates a Document Object Model of the page.
2. The HTML DOM model is constructed as a tree of Objects.

DOM:

Document Object Model (DOM) is a platform and language-neutral interface that allows programs and scripts to dynamically access and update the content, structure, and style of a document.

HTML-DOM:

The HTML DOM is a standard object model and programming interface for HTML. It defines:

1. The HTML elements as objects
2. The properties of all HTML elements
3. The methods to access all HTML elements
4. The events for all HTML elements

We can also say, The HTML DOM is a standard for how to get, change, add, or delete HTML elements.
